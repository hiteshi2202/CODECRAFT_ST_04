# Cross-Browser Testing with BrowserStack

## Objective
To perform automated cross-browser testing using Selenium on BrowserStack.

## Tools Used
- Python
- Selenium
- BrowserStack

## Test Scenario
Tested login functionality of Swag Labs website.

## Test Steps
1. Open Swag Labs website
2. Enter valid username and password
3. Click login button
4. Verify successful login

## Browsers Tested
- Google Chrome
- Mozilla Firefox
- Microsoft Edge

## Results
- Test executed successfully on all browsers
- No major issues found

## Conclusion
Cross-browser compatibility is maintained for login functionality.
